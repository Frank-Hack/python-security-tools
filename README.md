# 🐍 Python Security Tools

Repositorio con scripts de automatización y análisis orientados a ciberseguridad.

Los scripts están diseñados con fines educativos y para ser utilizados en entornos controlados de laboratorio.

---

## 🔧 Funcionalidades actuales

- Parser de resultados Nmap
- Automatización de tareas de reconocimiento
- Análisis de logs
- Scripts OSINT básicos

---

## 📁 Estructura

recon/ → scripts de reconocimiento  
osint/ → herramientas OSINT  
logs/ → análisis de registros  
utils/ → utilidades compartidas  

---

## ⚠️ Aviso

Este repositorio es únicamente con fines educativos y de laboratorio.  
No debe utilizarse contra sistemas sin autorización.
